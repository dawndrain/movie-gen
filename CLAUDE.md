# CLAUDE.md

Before any film work, read **MOVIE_LESSONS.md** — the single playbook
(pipeline, laws, prompting, voices, mixing, batches, costs).
**PROJECT_LOG.md** is the append-only project log: dated per-project
postmortems, newest first — the evidence behind the playbook's rules.

The protocol: lessons learned on a project go into PROJECT_LOG.md as a dated
addendum at the top, war-story detail and all. When a lesson generalizes
beyond its project, promote it into the right section of MOVIE_LESSONS.md —
and if it contradicts a rule already there, rewrite the rule in place rather
than appending a correction. There is deliberately only one playbook; don't
create parallel lessons docs.
